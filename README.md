# GlowQueen — Header Spa & Beauty

Recreación pixel-perfect de un hero section para una página web de spa y belleza, desarrollada como primera entrega del máster fullstack de Conquer Blocks.

## 🛠️ Tecnologías usadas

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![SASS](https://img.shields.io/badge/SASS-CC6699?style=flat&logo=sass&logoColor=white)

- **HTML5** — estructura semántica con etiquetas correctas (`<header>`, `<nav>`, `<main>`, etc.)
- **CSS3 / SASS** — layout con Flexbox, variables CSS, estados hover
- **SVG** — iconos vectoriales inline para mejor rendimiento y escalabilidad
- **Figma** — diseño original proporcionado por Conquer Blocks, usado como referencia

---

## ✨ Lo que incluye

- Barra de navegación con logo, enlaces y botones de acción
- Hero section con titular, descripción y CTAs ("Book Now" / "Watch Video")
- Estadísticas con iconos SVG (15+ tratamientos, 25+ terapeutas, 100% fórmula herbal)
- Tarjetas de testimonios flotantes con citas de clientes
- Estados hover en todos los elementos interactivos
- Fidelidad visual al diseño original de Figma
